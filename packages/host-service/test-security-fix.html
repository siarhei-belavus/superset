ℹ️ Vulnerability Details (SS-004)

Original Issue:

  • Unrestricted CORS (Access-Control-Allow-Origin: *)
  • Zero authentication on all endpoints
  • Exposed sensitive data (GitHub identity, Superset account)
  • Allowed destructive operations (PR merge, repo deletion)

Fix Applied:

  • ✅ Strict CORS (localhost/Electron only)
  • ✅ Session token authentication required
  • ✅ Constant-time token comparison
  • ✅ Secure token distribution via IPC
